Add Release
\r\n\t\t\t\t
\r\n\t\t
\r\n
\r\nAll releases will be verified before they get added to the website. For the \"Link to Release\" input, you may link to the table of contents or the translator's website if no direct link to chapter is available. No direct links to file sharing websites (DropBox, MediaFire, Mega, etc...) please.\r\n
\r\n
\r\nReminder: Only Asian translated novels are accepted at this time. If your releases aren't being accepted, please read NU's Policy on accepting/rejecting releases.\r\n\n
\n
\nTips\n
\n
\n
    \n
  • If the series or group name is not shown in the list, please add them first using add series or add group
    • \n
  • If you are adding old releases, please add the oldest chapter first. This is normally chapter 1 (chapter 0 if it exists). So you should add chapter 1 first, then chapter 2, etc...
